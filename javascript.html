Home Über mich Defend Humanity für iOS JavaScript

Um meine Fähigkeiten mit JavaScript zu zeigen, sind mir zwei Features auf dieser Webseite eingefallen, die ich auch eingebaut habe.

Einmal wäre da der Dunkel-Modus, den Sie durch klicken auf den oben rechts platzierten Knopf aktivieren bzw. deaktivieren können. Ihre Einstellung sollte im LocalStorage gespeichert werden, d.h. wenn Sie den Browser schliessen würden, sollte die Seite im zuletzt verwendeten Modus gestartet werden. Der Standard-modus ist der Dunkelmodus, wenn Sie also die Seite zum ersten Mal öffnen, wird der Dunkel-Modus geladen.

Das zweite Feature ist der Knopf auf der Home-page, welcher Sie zu einer zufällig ausgewählten Seite führt. Dieses Feature sollte selbsterklärend sein, daher habe ich dazu nicht viel zu erzählen, ausser, dass ich beachtet habe, dass man nicht auf die Home-page geschickt wird, weil das unnötig wäre.

Ich weiss, dass diese Features nichts bahnbrechendes sind, aber ich wollte auch nicht zu viel auf diese Seite stellen, da Sie mit Sicherheit nicht viel Zeit zur Verfügung haben.
